发表于 2019-07-12 更新于 2019-10-15 分类于 linux ， program

必装软件

sudo

sudo是linux系统管理指令，是允许系统管理员让普通用户执行一些或者全部的root命令的一个工具。必装。

AUR助手

AUR助手支持从安装AUR（ArchLinux User-community Repository）自动化安装软件，使用方式与pacman类似。常用的AUR助手有yay、yaourt、pacaur、pikaur、aurman。这里笔者安装的是pikaur。

安装AUR助手首先需要配置archlinuxcn中文社区仓库。

米聊

HexChat

常用软件

Terminator

好用的Linux终端。

uGet

跨平台、开源的多线程下载软件。

配合aria2使用

如果要使用浏览器扩展，还需安装uget-integrator

p7zip

”7-zip“的Linux版本，一款压缩解压工具。

Thunderbird

Mozilla开发的开源邮件客户端。

编程开发

开发语言/编译工具

GCC

GCC原名GNU C Compiler，因为它原本只能处理C语言。GCC很快地扩展，变得可处理C++。后来又扩展能够支持更多编程语言，如Fortran、Pascal、Objective-C、Java、Ada、Go以及各类处理器架构上的汇编语言等，所以改名GNU Compiler Collection。

Clang

C、C++、Objective-C的轻量级编译器。

CMake

一个跨平台的安装（编译工具）。

Node.js

Node.js是一个基于Chrome V8引擎的JavaScript运行环境。这东西让Javascript焕发了新生。

安装npm，npm是JavaScript世界的包管理工具,并且是Node.js平台的默认包管理工具。

Python

python3和python2

OpenJDK

开源的JDK，由Redhat社区维护。稳定性不及Oracle JDK，但是它免费啊！Oracle JDK说不定什么时候就收费了。Java EE已经改名叫Jakarta EE已经说明很多问题了。

文本编辑器

Vim

Linux上的著名文本编辑器vi的加强版。系统内置的好像不完整，需要自己更新。

VSCode

VScode，微软出品的轻量级文本编辑器。开源、免费、跨平台、易扩展、功能强大、性能优良。爱死这破软件了，自从装了这软件后再也没用过其他的图形文本编辑器。这不是在打广告！

IDE

Intellij Idea

IDEA 全称 IntelliJ IDEA，是java编程语言开发的集成环境。IntelliJ在业界被公认为最好的java开发工具之一，尤其在智能代码助手、代码自动提示、重构、J2EE支持、各类版本工具(git、svn等)、JUnit、CVS整合、代码分析、 创新的GUI设计等方面的功能可以说是超常的。旗舰版是要收费的，要是有教育邮箱的话可以免费订阅一年。

  • 社区版：

  • 旗舰版：

Code::Blocks IDE

知名开源跨平台的C/C++轻量级IDE。其基本功能VSCode也可满足，不过显然CodeBlocks更专业。

数据库

MariaDB

著名数据库MySQL的分支，由开源社区维护。自从MySQL被Oracle收购后，很多知名企业转向了MariaDB，不得不说Oracle让人又爱又恨。

DBeaver

开源的数据库管理工具。

Web服务器

Tomcat

Tomcat 服务器是一个免费的开放源代码的Web 应用服务器，属于轻量级应用服务器，在中小型系统和并发访问用户不是很多的场合下被普遍使用，是开发和调试JSP 程序的首选。

虚拟机

VirtualBox

开源、跨平台、免费的虚拟机软件。

其他

Git

git代码管理工具。

Devhelp

没用过。。据说可以查看GTK的开发文档。

办公软件

LibreOffice

开源、自由、跨平台的免费办公套件。
稳定版：

尝鲜版（beta版）：

Gimp

Gimp全名GNU Image Manipulation Program，是一款跨平台、自由的图像处理程序，功能丰富，号称“Linux下的PhotoShop”。

Inkscape

矢量图形编辑软件。我没用过，但是这个软件的好评比较高。

科学研究

Celestia

开源3D天文软件。

Chemtool

一款在Linux上绘制化学结构的软件。

Stellarium

Stellarium 是一款虚拟星象仪的计算机软件。它可以根据观测者所处的时间和地点，计算天空中太阳、月球、行星和恒星的位置，并将其显示出来。它还可以绘制星座、虚拟天文现象（如流星雨、日食和月食等)。

影音娱乐

mpv

Linux下最好用的视频播放器之一。支持绝大多数视频格式、功能强大、快捷键丰富。

Netease Cloud Music

网易云音乐。网易和Deepin公司联合出品。

VMPK

弹钢琴的软件。类似于一款小游戏，没有内置曲谱，无聊的时候可以用来放松一下。

其他工具

fish

好用的命令行提示工具。比内置的bash好用很多。

BaiduNetdisk

百度网盘。官方出品，感觉比其他版本的速度快很多，不知道是不是错觉。

w3m

运行在终端的开源网页浏览器。仅支持文本阅读，不需要图形界面即可运行。

you-get

下载youtube、优酷、B站视频工具。

Rarcrack

暴力破解压缩文件的密码。好像不支持跑字典。
