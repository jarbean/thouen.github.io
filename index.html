复杂度分析的主要方法

迭代式算法

针对迭代式算法我们主要采用级数求和的方法。级数求解的规律如下：

算术级数

算术级数在阶次上末项平方同阶。
$T(n) = 1+2+…+n = n(n+1)/2 = O(n^2)$

幂方级数

幂方级数在阶次上比幂次高出一阶。
$T_2(n) = 1^2+2^2+3^2+…+n^2 = n(n+1)(2n+1)/6 = O(n^3)$
$T_3(n) = 1^3+2^3+3^3+…+n^3 = n^2{(n+1)}^2/4 = O(n^4)$
$T_4(n) = 1^4+2^4+3^4+…+n^4 = n(n+1)(2n+1)(3n^2+3n-1/30 = O(n^5)$
…

几何级数

几何级数在阶次上与末项同阶。
$T_a(n) = a^0+a^1+a^2+…+a^n = (a^{n+1}-1)/(a-1) = O(a^n)$
$2^0+2^1+2^2+…+2^2 = 2^{n+1}-1 = O(2{n+1}) = O(2^n)$

收敛级数

收敛级数从渐进意义上来讲都是常数，都可以记作$O(1)$。
$1/(1*2)+1/(2*3)+1/(3*4)+…+1/[(n-1)*n] = 1-1/n = O(1)$
$1+1/2^2+1/3^2+…+1/n^2 = \pi^2/6 = O(1)$
$1/3+1/7+1/8+1/15+1/24+1/26+1/31+1/35 = 1 = O(1)$

递归式算法

针对递归式算法采用递归跟踪和递推方程方法。

排序算法

发表于 2019-07-19 更新于 2019-07-20 分类于 算法

数据结构常用的八大排序算法分别是：选择排序、冒泡排序、插入排序、快速排序、归并排序、基数排序、希尔排序和堆排序。以下对这八种排序算法的性能进行比较：

排序算法 时间复杂度 空间复杂度 稳定性
选择排序 $O(n^2)$ $O(1)$ 不稳定
冒泡排序 $O(n^2)$ $O(1)$ 稳定
插入排序 $O(n^2)$ $O(1)$ 稳定
快速排序 $O(n\log_2{n})$ $O(1)$ 不稳定
归并排序 $O(n\log_2{n})$ $O(1)$ 稳定
基数排序 $O(d(r+n))$ $O(rd+n)$ 稳定
希尔排序 $O(n^{1.3})$ $O(1)$ 不稳定
堆排序 $O(n\log_2{n})$ $O(1)$ 不稳定

基数排序的复杂度中，r代表关键字的基数，d代表长度，n代表关键字的个数。
