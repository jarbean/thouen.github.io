Archlinux桌面版安装教程

Posted on 2019-07-11 Edited on 2019-07-15

制作启动盘

下载系统

你可以从清华大学开源镜像站、中科大开源镜像站或者选择其他镜像源下载最新的ArchLinux系统。
下载完成之后你还可以使用Md5或sha1方式校验一下。

使用u盘制作启动盘

Linux平台

确定U盘是否挂载成功。

开始制作：if是你下载的archlinux系统iso文件路径，of是你的u盘位置，一般情况下为/dev/sdb。一定要等到sync完成，所有数据写入之后再拔掉u盘。

Windows平台

建议使用Rufus或USBwriter制作启动盘。不要用UtralISO（软碟通），因为没有用。

启动电脑并配置BIOS

开机或重启 -> 打开BIOS -> 设置允许电脑从u盘启动。

安装系统

进入安装环境

电脑从u盘启动后，看到Arch菜单出现，选择Boot Arch Linux，点击Enter进入安装环境。屏幕上如果报错请仔细检查BIOS配置。

联网
