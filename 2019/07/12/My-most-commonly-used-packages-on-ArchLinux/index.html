我的Archlinux常用软件包

发表于 2019-07-12 更新于 2019-07-21 分类于 Archlinux

Archlinux官方仓库

CMake

一个跨平台的安装（编译工具）。

Celestia

开源3D天文软件。

Chemtool

一款在Linux上绘制化学结构的软件。

Code-OSS

VScode，微软出品的轻量级文本编辑器。开源、免费、易扩展。

Code::Blocks IDE

知名开源跨平台的C/C++轻量级IDE。

Dconf Editor

gnome高级配置。

Devhelp

查看开发文档。

Fcitx

fcitx输入法。

Firefox

火狐浏览器。一个跨平台、开源的著名网页浏览器。

Gnome

Gnome桌面环境。Gnome是一款开源、易扩展的Linux桌面环境。

Gnome Tweaks

Gnome美化、优化工具。

Gimp

Gimp全名GNU Image Manipulation Program，是一款跨平台、自由的图像处理程序，功能丰富，号称“Linux下的PhotoShop”。

GParted

Linux磁盘分区工具

Vim

Linux上的著名文本编辑器vi的加强版。

gVim是Vim的图形前端，有图形界面需求的可以安装gVim。

HexChat

支持多语言的跨平台、开源IRC客户端，用来聊天。

LibreOffice

开源、自由、跨平台的免费办公套件。
稳定版：

尝鲜版：

Oracle VM VirtualBox

开源、跨平台、免费的虚拟机软件。

Seahorse

用于管理GnuPG密钥。

Stellarium

Stellarium 是一款虚拟星象仪的计算机软件。它可以根据观测者所处的时间和地点，计算天空中太阳、月球、行星和恒星的位置，并将其显示出来。它还可以绘制星座、虚拟天文现象（如流星雨、日食和月食等)。

Terminator

终端终结者。

Thunderbird

Mozilla开发的开源邮件客户端。

VMPK

弹钢琴软件。

uGet

跨平台、开源的多线程下载软件。

GCC

GCC编译器，C/C++开发必备。

OpenJDK

开源和自由的JDK，由社区维护，稳定性不及Oracle JDK。

MariaDB

著名数据库MySQL的分支，由开源社区维护。自从MySQL被Oracle收购后，很多知名企业转向了MariaDB。

Maven

Java项目管理工具。

Node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。

npm

npm 是 JavaScript 世界的包管理工具,并且是 Node.js 平台的默认包管理工具。

p7zip

”7-zip“的Linux版本，一款压缩解压工具。

sudo

sudo是linux系统管理指令，是允许系统管理员让普通用户执行一些或者全部的root命令的一个工具。

Tomcat

Tomcat 服务器是一个免费的开放源代码的Web 应用服务器，属于轻量级应用服务器，在中小型系统和并发访问用户不是很多的场合下被普遍使用，是开发和调试JSP 程序的首选。

VirtualBox

VirtualBox号称是最强的免费虚拟机软件。

w3m

w3m是个开放源代码的文字式网页浏览器。

AUR仓库

安装AUR（ArchLinux User-community Repository）的软件，首先要安装yaourt、yay或者pikaur。这里推荐pikaur。

BaiduNetdisk

百度云盘。

Intellij Idea

IDEA 全称 IntelliJ IDEA，是java编程语言开发的集成环境。IntelliJ在业界被公认为最好的java开发工具之一，尤其在智能代码助手、代码自动提示、重构、J2EE支持、各类版本工具(git、svn等)、JUnit、CVS整合、代码分析、 创新的GUI设计等方面的功能可以说是超常的。
社区版：

旗舰版：

Mojava-GTK-Theme

Mac OSX风格的桌面主题。

Netease Cloud Music

网易云音乐。

Rarcrack

暴力破解压缩文件的密码。
